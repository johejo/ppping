# ppping

## Description
Petty Plain Ping Tool like [deadman](https://github.com/upa/deadman)

## Demo
![result](https://github.com/johejo/ppping/blob/master/demo.gif)

## Install

```bash
$ pip install ppping
```

# Usage

```bash
$ ppping foo.com bar.org WW.XX.YY.ZZ
```
Future help

```bash
$ ppping --help
```
